## 财务管理系统（系统分析与设计课程期末作业）

---

##  项目简介

**Accounting Management System** 是一个基于**复式记账原理**设计与实现的财务管理系统，  
主要面向**个人用户与中小型企业**，实现从财务数据记录、分类管理到报表分析的完整流程。

系统以 **资产、负债、权益、收入、支出** 五大会计科目为核心，  
支持规范化记账、自动生成财务报表，并引入 **AI 财务助手**，  
为用户提供智能化的财务分析与问诊能力。

本系统在设计上参考了 **GnuCash** 的会计模型，  
在保证专业性的同时，兼顾系统的易用性、可扩展性与教学实践价值。

##  技术架构（Technology Stack）

### 后端
- **Spring Boot** —— 系统核心框架  
- **MyBatis / MyBatis-Plus** —— ORM 持久层  
- **RESTful API** —— 前后端通信  
- **MySQL** —— 关系型数据库  

### 前端
- **HTML5 / CSS3**
- **原生 JavaScript**
- **ECharts**（财务数据可视化）
